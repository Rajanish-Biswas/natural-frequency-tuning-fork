Bearing Defect Frequencies — DM Pump-3 (Approximate, 1475 RPM)

Actual bearing defect frequencies depend on bearing geometry (number of balls, contact angle, pitch diameter, ball diameter). The values below use typical ratios for medium-duty pump bearings. For exact values use bearing manufacturer data (SKF, FAG, NSK).

Frequency Name Abbr. Typical Ratio Approx. Hz (at 1475 RPM) Fault indicated
Fundamental Train Frequency FTF ~0.4× running speed ~9.8 Hz Cage defect / cage slip
Ball Pass Frequency Outer race BPFO ~3.6× running speed ~88.5 Hz Outer race pitting / spalling
Ball Pass Frequency Inner race BPFI ~5.4× running speed ~132.7 Hz Inner race pitting / spalling
Ball Spin Frequency BSF ~2.4× running speed ~59.0 Hz Ball / rolling element defect

⚠ Bearing defect frequencies are non-synchronous — they are NOT integer multiples of running speed. This is what distinguishes them from unbalance/misalignment/looseness on an FFT spectrum. Elevated spike energy (gSE) is often the first indicator — before these frequencies appear clearly on the spectrum.
